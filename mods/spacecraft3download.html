Spacecraft3.dll, Spacecraft4.dll

Orbiterに宇宙船などを追加するためのMODです。

ダウンロード

Pappy's Hangar - Spacecraft4※

このMODには、Spacecraft、Spacecraft2、Spacecraft3、Spacecraft4の4つのdllファイルが収録されています。

インストール

ダウンロードしたファイルを解凍して、中身をすべてOrbiterフォルダに上書き。

操作方法

操作方法については以下のページを参照してください。
Spacecraft3、Spacecraft4のどちらも同じように操作できます。

Spacecraft3 キー操作一覧

Multistage2015.dll, stage.dll

複数のロケットを組み合わせて、多段式ロケットを再現するためのMODです。

ダウンロード

以下のファイルには、stage.dllも同梱されています。

Orbiter2010対応版
Multistage2015 - for Orbiter 2010P1

Orbiter2016対応版
Multistage2015 - for Orbiter 2016

インストール

ダウンロードしたファイルを解凍して、中身をすべてOrbiterフォルダに上書き。

Modulesタブを開いて、Multistage2015_MFDにチェックを入れる。

Multistage2に適用する

一部の古いMODでは、Multistage2015ではなくMultistage2が必要なものがあります。
オリジナルのMultistage2はサイト閉鎖により現在公開されていません。

Multistage2015をMultistage2に適用するための手順は以下の通り。

Orbiter\Modulesフォルダを開く。
Multistage2015.dllをコピーして、片方をMultistage2.dllにリネーム（名前を変更）する。

Orbiter\Configフォルダを開く。
multistage2015.cfgをコピーして、片方をmultistage2.cfgにリネーム（名前を変更）する。

新しく作成したmultistage2.cfgをメモ帳などのテキストエディタで開く。
以下のように書き換えて上書き保存する。 

ClassName = Multistage2
Module = Multistage2

ConfigフォルダにMultistageという名前の空フォルダを作成する。
multistage2.cfgをコピーして、Multistageフォルダに貼り付ける。

シナリオに追加する

Spacecraft.dllを利用した機体を、シナリオエディタを使ってそのまま追加することはできません。
シナリオファイルを直接編集するか、機体ごとにcfgファイルを作成する必要があります。

シナリオファイルの直接編集

cfgファイルを作成する

あらかじめcfgファイルを作成しておくと、ゲーム内からシナリオエディタを使って宇宙船を追加できます。

メモ帳などのテキストエディタを開く。

ClassName = ファイル名（機体のiniファイル名と同じ）
Module = Spacecraft3

以上のように書いたファイルを作って、拡張子を.cfgにする。（MODの名前.cfgで保存する）

このファイルを、Orbiter\Config\Vesselsに置く。

ゲーム内でシナリオエディタを開いて、Newをクリック。
Vessel typeとVessel nameを同じにしてCreateをクリック。
