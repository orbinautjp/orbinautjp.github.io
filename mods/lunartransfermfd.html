特徴

LunarTransferMFDは、地球から月へのフライトのためのMODです。
地球→月における軌道予測の精度が高く、Delta Vの制限が厳しいミッションで活躍します。

地球とその月でしか使えません。
イオンエンジンなどはサポートしていません。
作成できる軌道には制限があります。

ダウンロード

リンク先ページ右上のDownloadボタンをクリック。

どちらのバージョンでも、同じように操作できる。

Orbiter2010

Orbiter2016

  • LunarTransfer MFD (LTMFD) 1.6

インストール

ダウンロードしたファイルを解凍して、中身をOrbiterのフォルダに上書き。

LaunchpadのModulesタブを開く。
LunarTransferMFDにチェックを入れる。

操作方法

基本的な操作は、IMFDとほぼ同じ。

AMSO

  • アポロ11号チュートリアル

キー操作一覧

PRG Program menuを開く
PRVorNXT ほかの項目へ移動
+or-
SET		 入力・選択
ADJ 倍率の変更
EXE 設定項目の決定・実行
DV エンジン噴射のための画面を開く
VM 画面の変更（月通過・地球最接近などを切り替えられる）

地球―月遷移軌道に乗る

地球軌道上から月へ向かう場合は、以下のような手順になる。

PRGをクリックしてProgram menuを開く。

Program TLIを選択。
TIg（出発時刻）、PeT（到着時刻）などを入力する。
EXEをクリック。

DVをクリックしてDelta Velocity画面を開く。
EXEをクリック。

自動噴射のカウントダウンが開始される。
