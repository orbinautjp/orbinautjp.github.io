宇宙船を選択する

シナリオエディタを開くと、そのシナリオに含まれる宇宙船が一覧表示される。
宇宙船をクリックして選択する。

右下のEditをクリック。
軌道要素や現在位置、残り燃料などを変更する。

以下の3つのカテゴリーから、変更したい項目を選ぶ。

Free flight 自由落下
On the ground 地上
Misc その他

設定が終わったら、ウィンドウ下のApplyをクリック。
変更が適用される。

Refreshをクリックすると、時間経過による変化が反映される。

Free flight

軌道要素や角速度など、宇宙船の軌道上での状態を決定する。

Orbital elements

宇宙船の軌道要素を決定する。

Semi-major axis [SMa]

軌道長半径。楕円軌道の長半径を指定する。
地球のまわりをほぼ完全な円軌道で周回する場合、6571kmなら高度200kmになる。

Eccentricity [Ecc]

軌道離心率。0にするとほぼ完全な円軌道になる。

Inclination [Inc]

軌道傾斜角。基準となる平面（黄道もしくは赤道）からの傾きを指定する。

Longitude of ascending node [LAN]

昇交点黄経／赤経。軌道の傾きは同じままで、向きが変わる。
軌道傾斜角が同じでも、LANが違えば軌道面は一致しない。

Orbit reference

基準となる天体（中心天体）を選択する。

Frame

基準となる平面を選択する。

黄道面を基準にするか、赤道面を基準にするか決める。
Orbit MFDなどを使用しているときは、同じ基準を参照するように設定する。

ecliptic 黄道面を基準にする
ref. equator 赤道面を基準にする

Epoch

元期を決定する。
「この時刻に、左のような軌道要素だった」ことを表す。

current （ゲーム内の）現在時刻
MJD 修正ユリウス日による任意の日時

打ち上げを省略する

打ち上げを省略して、ISSと同じ軌道面に宇宙船を配置することができる。

Orbit MFDを開いてISSをターゲットする。
Orbit MFDの右側を見て、ISSのIncとLANを確認する。

シナリオエディタを開く。

Frameをref. equatorにする。
ISSと同じ数値をIncとLANに入力する。
Semi-major axisを6571km、 Eccentricityを0にする。

Applyをクリックすると、ISSと軌道面が一致した状態になる。

State vectors

軌道状態ベクトルを決定する。
初期状態ではcartesian（デカルト座標）で表現される。

\( ( x, y, z ) \) 位置ベクトル
\(\displaystyle \left( \frac{dx}{dt}, \frac{dy}{dt}, \frac{dz}{dt} \right) \) 速度ベクトル
Orbit reference 中心天体
Frame 基準となる平面
Coordinates 座標系
Copy state from 軌道状態ベクトルをコピーする

Coordinatesをpolar（極座標）にすると、表示が変化する。

radius 動径
longitude 経度
latitude 緯度
\(\displaystyle \left( \frac{ d~ \mathrm{ radius } }{dt} \right) \) 動径方向の速度。経度・緯度についても同じ。

Copy state from

Ctrl+Pを押して、ゲームを一時停止する。
Copy state fromのリストから、任意の宇宙船をクリック。
Applyをクリックすると、選択した宇宙船と同じ位置・軌道に移行する。

Orientation

機体の向きを指定する。
地上では変更できない。

Angular velocity

角速度。
宇宙船が回転する速度を変更できる。

Killをクリックすると回転が止まる。
宇宙船が高速回転して制御できなくなった場合でも、強制的に止めることができる。

On the ground

地上での位置を変更する。

Location

地上にいる宇宙船の位置を変更する。
軌道上にいる宇宙船を地上に降ろすこともできる。

Surface position

地上での位置を決定する。

Longitude 経度
Latitude 緯度
Heading 宇宙船の向き

Reference

惑星や月などの天体を選択する。
選択した天体に移動する。

Celestial body 天体（惑星とその月）の選択
Surface base 地上基地
Landing pad 着陸パッド

滑走路を指定することはできない。

Copy state from

地上にいる他の宇宙船から、位置情報をコピーする。

これを使うと、他の宇宙船と同じ位置に移動する。
Orbiterには宇宙船どうしの衝突判定はないので、2機が重なり合うことになる。

Misc

その他の項目。

Propellant

推進剤の残量を変更する。

Tank no. 推進剤タンクの番号
Empty 空にする
Full 満タンにする
Empty all 全タンクを空にする
Fill all 全タンクを満タンにする

Tank no.

どの推進剤タンクの残量を変更するか選択する。

宇宙船によっては、スクラムジェット、メインエンジン、RCSで別のタンクを使用することがある。
逆に、探査機・人工衛星では、すべて同じタンクの燃料を使用するものもある。

Docking

宇宙船および宇宙ステーションとドッキングする。

相手の宇宙船とドッキングポートを選んで、Dockをクリック。
即座にドッキングされる。

Dock no. 自機のドッキングポートを選択
Establish docking connection with ドロップダウンメニューで宇宙船を選択
using dock 相手のドッキングポートを選択
Bring target to... 相手がこちらの現在位置に移動する
Move me to... こちらが相手の現在位置に移動する
