シナリオエディタ 宇宙船の追加

シナリオエディタを使用して、ゲーム内で宇宙船を追加・削除する。

  • 宇宙船を選択する
  • 宇宙船を削除する
    • ゲームが強制終了するとき
  • 宇宙船を追加する
  • その他のケース

宇宙船を選択する

シナリオエディタを開く。
宇宙船の名前をクリックして選択する。

右上のTrackにチェックが入っていると、選択した宇宙船にカメラが移動する。

宇宙船を削除する

リストから選択して、右下のDeleteをクリックすると、既存の宇宙船を消去できる。

ゲームが強制終了するとき

何らかの不具合でゲームが強制終了する場合、原因になっている宇宙船を削除すると復旧できることがある。

宇宙船を追加する

Newをクリック。

リストから宇宙船を選んでVessel name:に名前を入力。
左下のCreateをクリックすると、宇宙船が追加される。

右上のSet cameraにチェックが入っていると、追加した宇宙船にカメラが移動する。
Set input focusにチェックを入れておくと、追加した宇宙船をそのまま操作できる。

その他のケース

Spacecraft.dllを利用する宇宙船は、あらかじめConfigファイルを作成しないとリストに表示されない。

複数のモジュールが結合した宇宙船・宇宙ステーションを追加する場合は、シナリオファイルを直接編集する。
