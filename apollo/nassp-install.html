NASSPのダウンロード+インストール

Project Apollo - NASSPは、アポロ宇宙船・サターンロケットのMODです。
実機のシステムを再現しており、コクピットのボタン類も実際に操作できます。

ダウンロード

下記のリンクを開く。
ページ右上のリンクをクリック。

Installation - ProjectApollo

Orbiter2016 Beta NASSP 8.0 以上
Orbiter2010 NASSP 7.0

8.0は Orbiter 2016 Betaが必要。
普通のOrbiter2016では動かない。

インストール

ダウンロードしたファイルを解凍する。
中身をOrbiterのインストールフォルダに上書き。

LaunchpadのModulesタブを開く。
以下の2つのモジュールにチェックを入れる。

  • ProjectApolloMFD
  • RTCC MFD

Visual C++のランタイムを導入していない場合は、以下のファイルもインストールする。

Microsoft Visual C++ 2015 再頒布可能パッケージ

ダウンロードしたファイルをダブルクリック。
インストーラの指示に従ってインストールする。

設定

LaunchpadのParametersタブを開く。
以下の項目にチェックを入れる。

  • Complex flight model
  • Limited fuel
  • Gravity-gradient torque
  • Nonspherical gravity sources

設定項目の意味については、以下のページを参照すること。

Realism（リアリズム）とPerturbations（摂動）の設定
