NASSPのダウンロード+インストール

Project Apollo – NASSPは、アポロ宇宙船をOrbiter上で再現したMODです。
コクピットのスイッチ類を実際に操作してアポロ宇宙船を操作することができます。

現時点では、Orbiter2016には対応していません。（Orbiter2010のみ）

ダウンロード

Installation – ProjectApollo

ページ右上のリンクをクリックしてダウンロードする。

インストール

ダウンロードしたファイルを解凍して、Orbiterのインストールフォルダに上書き。

OrbiterにMODを導入する

LaunchpadのModulesタブで、ProjectApolloMFDにチェックを入れる。

Visual C++のランタイムを導入していない場合は、こちらもインストールしてください。

Microsoft Visual C++ 2015 再頒布可能パッケージ

ダウンロードしたファイルをダブルクリックして、インストーラの指示に従ってください。

設定

LaunchpadのParametersタブを開く。
以下の項目にチェックを入れる。

  • Complex flight model
  • Limited fuel
  • Gravity-gradient torque
  • Nonspherical gravity sources

設定項目の意味については、以下のページを参照してください。

Realism（リアリズム）とPerturbations（摂動）の設定
